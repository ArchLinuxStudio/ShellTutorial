在学习 Linux 命令行时，各个命令有大量的参数以及组合，没有必要死记硬背全部的参数和其作用，只需要记住常用的组合，以及在需要查询某一个用法的参数时，知道如何通过 man 等方式快速查询即可。对于 Bash 脚本编程，也是熟能生巧的内容，安装一个 Linux 系统，开始你的命令行与 Shell 编程之旅。
